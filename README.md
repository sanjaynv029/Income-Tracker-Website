# Income-Tracker-Website
The Income Tracker Website is a web-based application developed using Spring Boot and MySQL that helps users efficiently manage and monitor their personal finances. The system allows users to record, categorize, and analyze their income sources, providing insights into financial patterns and helping them make better budgeting decisions.

Overview of Project:
The project aims to ease the process of handling personal finance by providing tools like income tracking, expense management, budget planning, and analysis. The system enables users to make better financial decisions based on recorded data.

Technologies Used:
Backend: Java, Spring Boot
Frontend: HTML, CSS, JavaScript
Database: MySQL
IDE: Eclipse / IntelliJ IDEA
Key Features
Registeration and Secure Login
Creation, Modification, and Deletion of Income Details
Expense Tracking and Classification
Budget Management by Month
Finance Dashboard
Responsive Interface

Modules:
1. User Authentication Module
Ensures secure user registeration and login along with authentication and sessions management.
2. Income Management Module
Enables users to create and manage incomes such as salary, freelancer payments etc.
3. Expense Management Module
Lets users track their daily expenses and classify them into categories like food, travel, shopping, utilities, and entertainment.
4. Finance Reports Module
Provides users with graphical reports and finance statistics that help them analyze their transactions.

Objective:
The objective of this project is to create an effective and robust solution to manage personal finances digitally.
